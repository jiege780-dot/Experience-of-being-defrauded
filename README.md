# Experience-of-being-defrauded
掉入陷阱的开始
我正式接通诈骗电话开始被骗日期始于8月29号，但在这之前，我已经持续收到2XX开头显示来自华盛顿的电话几个月了。我还记得那天晚上我正独自呆在我表姐家的一个房间。这时手机上莫名其妙的又出现了几通来自华盛顿的电话，我内心不禁怀疑起来，这到底是个什么电话，难道真的找我有什么事。于是我关上了房门，接通了这起电话。（事实上从后来来看，在我产生疑问，接通电话的这一刻，我就已经彻底输了）
稍微安静了片刻，电话那头响起了一个声音。
“您好，这里是华盛顿大使馆，我是今天的值班主任，X主任，请问您是刘先生吗？”，
我答道是的，于是他又说“我现在用的是我值班室的电话打给你的，请您现在在网上搜索一下中国大使馆，并找到中国大使馆的官方电话，我用这个电话回给您。”
我搜索了中国大使馆，并看到了下面有一个号码，然后我挂断了电话。过了一会儿，他们果然用这个号码打给了我。那位所谓的大使又接着说道。
他们这边收到一起有关我护照问题的通知，问我这两天可不可以来一趟华盛顿大使馆，以便告知我发生了什么事情。“护照问题？还要我亲自去大使馆？我该怎么去呢？”
于是我说，我离大使馆这么远肯定没法去啊，我这两天看看能不能去吧。
对面听我这么说完，说“那既然你来不了大使馆，那我现在就通过电话告知你发生了什么事吧，你身边有没有纸和笔，把重要的信息记录下来。”我急忙说有，于是准备开始，
对面的“大使”讲到：“在8月16号的时候，我们接到广州市公安局的一起通知，这天他们在广州市白云机场逮捕了一名叫做陈军的女性，发现在他的行李箱里，有几十本假护照，其中一本就是我的。这件事情可能会影响到我护照的国际信誉，导致我不能正常出入境。所以希望我明天就来一趟大使馆进行面谈。
我慌了，去大使馆面谈这是要干嘛啊。我说我可能真的去不了，那我该怎么办呢。他就说推荐我直接报警，毕竟我过去了也不能怎么样，就是沟通一下，这件事不归大使馆管辖。然后他告诉了我这件事的案件编号，他的姓名，以及他的工号。
我说那我怎么打给国内警察报案呢，我人还在境外。
他回到：“那好吧，那这样，我帮你转接国内公安局，然后你告诉他们发生了什么。一定要清楚的告诉他们，并让他们给你出具一张“报警回执单”有了这个东西后你的护照就不会有任何问题了。然后有什么问题随时可以打这个电话联系我“
在一阵短暂紧张的转接等待后，那边响起了一个年轻男性的声音：“您好，我是今天的值班警察刘凯，请问有什么可以帮您的吗。”
我结结巴巴的说道：“我是一名在美国的留学生，我刚刚接收到了中国大使馆的电话。。。。，然后我告知了他们事情的缘由。
然后对面的警察说；“那如果您想要报案的话需要亲自来警察局啊。电话报案是没法立案的，也没法出具回执单”
挂断了电话，我不知道该怎么办才好，于是又回拨了那个“大使馆”的电话，这一来二去之间，我已经丧失了怀疑的能力，完全被牵着走
很快那边就接通了电话，“大使”问我情况怎么样了，我就如实汇报给了“大使”
“大使”震惊地说道，啊？你现在护照存在问题就出不了境啊，我再把电话给你转过去，你给他们说清楚，说你现在出不了境，必须要一个“报警回执单”
